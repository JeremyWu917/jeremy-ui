# Color 色彩

> `JeremyUI` 为了避免视觉传达差异，使用一套特定的调色板来规定颜色，为你所搭建的产品提供一直的外观视觉感受。

## 通常颜色

`JeremyUI` 主要品牌颜色是鲜艳、友好的紫色。
